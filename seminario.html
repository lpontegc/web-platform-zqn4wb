El objetivo de este trabajo consiste en hacer uso de las herramientas vistas en clase para crear una web que visualice información de un dataset elegido.

En este caso el dataset seleccionado se trata de un conjunto de datos que contiene información sobre el top 50 de los libros más vendidos en Amazon desde el año 2009 hasta el año 2019. Este dataset contiene la información mostrada a continuación:

  1. Nombre del libro
  2. Autor
  3. Rating
  4. Reviews
  5. Precio
  6. Año
  7. Género

Mediante esta información, se mostrarán diversas visualizaciones orientadas a un público lector que esté en busqueda de libros populares. De esta manera, podrán conocer más información sobre estos para así tomar su decisión a comprar uno de ellos. Para ello, se han utilizado las herramientas Tableau y DataWrapper

DATAWRAPPER

Como se puede observar en la imagen anterior, este gráfico nos muestra el top 15 de los libros con mayor precio entre los años 2009 y 2019.

TABLEAU DESKTOP

En este dashboard compuesto mediante la herramienta de Tableau Desktop podemos observar diferentes graficos.

El primero de ellos nos indica en una gráfica de lineas, la media de precio del top 50 de libros mas vendidos en Amazon por cada año. Esto nos permite ver como los precios de los libros más vendidos tienen una tendencia a ser más baratos a lo largo del tiempo

En este segundo gráfico podemos ver una distribución de los libros por precio y rating recibido, pudiendo ser filtrado por año de venta o precio de este. De esta manera, se puede observar la distribucion clara en base a precio y rating.
